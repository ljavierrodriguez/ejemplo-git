### Iniciar un proyecto Git

```shell
git init
```

### Preparar los archivos

```shell
git add <name>
```
```shell
git add -A
```

```shell
git add .
```

```shell
git add *
```

### Guardar cambios en el repositorio local

```shell
git commit -m "los cambios realizados"
```

### Ver informacion del o los repositorios remotos

```shell
git remote -v
```
